# Contract Analysis - Complete Visual Guide

## 🎬 User Journey

```
START
  ↓
[Home Page] → Click "Contract Analysis" button
  ↓
[Upload Page] → Select PDF contract
  ↓
(Optional) → Add title, notes, select focus areas
  ↓
[Click "Analyze Contract"]
  ↓
⏳ ANALYZING (1-3 minutes)
  ├─ Loading animation
  ├─ "Analyzing Document" message
  └─ Progress indicators
  ↓
✅ ANALYSIS COMPLETE
  ↓
[Results Screen] → Two tabs appear:
  ├─ 📋 FULL REPORT (default)
  │  └─ 16 detailed sections
  │
  └─ 📊 EXECUTIVE SUMMARY
     └─ Quick overview + recommendations
  ↓
[Q&A Chat Enabled]
  ├─ Ask questions
  ├─ Get answers
  └─ Export results
  ↓
END
```
